Class XML

An

  1. XML
object represents XML content and provides methods to manipulate it.

The

  1. XML
object has an 'active' element, which represents the hierarchy at which the XML can be manipulated. <p> As an example, one
  1. creates a XMLBuilder object.
  2. create a configuration as an XML object.
  3. Call the loadXMLConfiguration(XML) method on Device

  1. only element: Append an element under the active element
of XML object. The new element now becomes teh active element.
  2. one element and one text : Append an element, with text, under the

the active element.
  3. one element and an arrya of text values : Append multiple elements,

key value.
